## 📝 如何使用 Examples

### 1. 阅读 + 学习
按推荐路径阅读，理解每个示例展示的风格元素和分析框架应用。

### 2. 复用模板
每个示例的"框架应用模板"或"输出结构"部分可直接复制使用。

### 3. 风格校准对比
将你的 AI 模拟输出与示例对比，按 [`references/07-checklist.md`](../references/07-checklist.md) 打分。

### 4. 测试 prompt
用 [`scripts/test-prompts.json`](../scripts/test-prompts.json) 中的 prompt 测试你的 AI Agent，看输出是否匹配示例风格。

### 5. 贡献新 Example
参考 [`CONTRIBUTING.md`](../CONTRIBUTING.md) 提 PR。
